🎮 Flappy Bird AI Agent (Reinforcement Learning)

An intelligent game-playing agent that learns to master Flappy Bird using Reinforcement Learning—without any hardcoded rules. The agent improves its performance over time by interacting with the environment and optimizing decisions to achieve higher scores.

🚀 Why This Project Stands Out
🧠 Built using Deep Reinforcement Learning (DQN)
🎯 Solves a real-time decision-making problem
📈 Demonstrates learning from scratch (no supervision)
⚡ Shows clear performance improvement over training episodes
🎮 Combines Game Development + AI in a practical scenario
🧠 Key Concepts Implemented
Markov Decision Process (MDP)
Q-Learning / Deep Q-Network (DQN)
Exploration vs Exploitation (ε-greedy strategy)
Reward Optimization
⚙️ How It Works

The agent observes the game state (position, velocity, pipe distance) and decides whether to flap or not.
It receives rewards based on survival and successfully passing obstacles, and gradually learns an optimal strategy through continuous interaction.

📊 Results
Agent learns to survive longer with training
Achieves significantly higher scores than random gameplay
Demonstrates stable and optimized decision-making over time
